QUESTION TAG

In grammar, a question tag is a very short clause at the end of a statement which changes the statement into a question. For example, in 'She said half price, didn't she?', the words 'didn't she' are a question tag.

Let's begin with some simple example

She's a doctor, isn't she?
[वह एक डॉक्टर है, है ना ?]


Yesterday was so much fun, wasn't it ?
[कल बहुत मज़ा था, है ना?]

He isn't here, is he?
[वह यहाँ नहीं है, है ना?]

The trains are never on time, are they?
[ट्रेनें कभी समय पर नहीं होती हैं, है ना?]

Nobody has called for me, have they?
[किसी ने मुझे नहीं बुलाया है, है ना?]

I don't need to finish this today, do I?
[मुझे इसे आज समाप्त करने की आवश्यकता नहीं है, है ना?]

James is working on that, isn't he?
[जेम्स उस पर काम कर रहा है, है ना?]

Defination

Tag questions (or question tags) turn a statement into a question. ... Usually if the main clause is positive, the question tag is negative, and if the main clause is negative, it's positive. For example: It's cold (positive), isn't it (negative)? And: It isn't cold (negative), is it (positive)?

Types of questino tag

  1. Positive Question Tags
  2. Negative Question Tags
